# CNC TP-AT Software
Реализация desktop приложения - <b>C#</b> <br>
Реализация прошивки микроконтроллеров на базе STM32 - <b>C</b> <br>
Протокол связи между ПО и микроконтроллерами - <b>Modbus RTU (USB интерфейс)</b> <br>
